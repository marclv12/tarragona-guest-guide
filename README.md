# Tarragona Central Apartment · Guest Guide

Pack de web estàtica per a hostes de Booking/Airbnb.

## Fitxers

- `index.html`: web completa, multilingüe i responsive.
- `README.md`: aquestes instruccions.

## Com provar-ho

1. Descomprimeix el ZIP.
2. Obre `index.html` amb doble clic.
3. Prova els botons d’idioma, Wi-Fi, mapa i WhatsApp.

## Com publicar-ho fàcilment

### Opció recomanada: Netlify
1. Entra a https://app.netlify.com/drop
2. Arrossega la carpeta sencera `tarragona_guest_guide_pack`.
3. Netlify et donarà un enllaç públic.
4. Aquest enllaç és el que pots posar en un QR.

### Opció GitHub Pages
1. Crea un repositori nou.
2. Puja l’arxiu `index.html`.
3. Activa GitHub Pages des de Settings > Pages.
4. Tindràs un enllaç públic.
